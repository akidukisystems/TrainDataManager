１．これはなに？
Minecraft 1.12.2、Forge 2860、最新版のRealTrainModを利用して、外部ソフトウェアから列車の挙動を制御、モニタリングするModです。

２．どうやって使うの？
このリポジトリをダウンロードし、gradlew-j8 build コマンドを実行してビルド、modsフォルダにいれてください。
ワールド読み込み直前に、mc2pi\srcフォルダ内のjavaファイルをコンパイル・実行して、接続してください。

３．なにが必要なの？
- Java 8 開発環境
- MinecraftのModding技術
- Javaの知識
- 時間と体力
- 余裕
