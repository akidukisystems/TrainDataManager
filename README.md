# これってなに？　What is this?
Minecraft JE 1.12.2、Real Train Mod、ATS Assist Mod、ズイキマスコン、Raspberry Pi 5、Arduino UNO R3（自作互換機）、HT16K33を利用して、JR東日本 E233系の運転台を模したタッチディスプレイを動作させるためのプログラムです。

# どうやって使うの？　How to use
他の方が利用することは一切想定しておりません。ソースコードは全てCC BY-SA 4.0として公開します。ご自由にご利用ください。

どうしても使いたい場合は、以下のコンポーネントを用意することをおすすめします。一切のサポートはできかねます。
+ Minecrat JE 1.12.2
+ Minecraft JE 1.12.2対応の、最新の1.12.2Minecraft Forge
+ Minecraft JE 1.12.2対応の、最新のNGTLib
+ Minecraft JE 1.12.2対応の、最新のReal Train Mod
+ Minecraft JE 1.12.2対応の、最新のATS Assist Mod
+ Zuiki Mascon（ワンハンドル、EB-B8-B1-N-P1-P5、ZKNS-013推奨）
+ Raspberry Pi 5 8GB以上モデル
+ Arduino（UNO R3以上推奨）
  + HT16K33をI2Cで接続することを推奨
+ タッチディスプレイ（HDMI入力、FHD 2枚接続を推奨）
+ org.json
+ jinput
+ WrapLayout
